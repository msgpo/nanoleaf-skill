## nanoleaf-skill
This skill permits mycroft the ability to control the NanoLeaf Aurora lighting
## Description 
Mycroft.AI will have control over the smart wifi dimmer switch
## Examples 
* "turn the aurora on"
* "turn the aurora off"
* "dim the aurora"
* "set the aurora to 25%"
* "set the aurora to red"
